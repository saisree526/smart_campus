Smart Campus Analytics Platform
Overview

The Smart Campus Analytics Platform is a data analytics project developed to help educational institutions monitor and analyze student academic performance and attendance. The system collects student information, stores it in a database, performs data analysis, and presents meaningful insights through an interactive dashboard.

The project demonstrates how data analytics can be used in educational environments to support better academic monitoring and decision-making.

Project Goal

The primary goal of this project is to transform raw student data into actionable insights that can help educational institutions:

Monitor student attendance and academic performance.
Identify students who may require additional academic support.
Analyze department-wise performance.
Improve decision-making using data-driven insights.
Reduce manual effort in report generation and data analysis.
Why This Project Is Used

Educational institutions often maintain student records in spreadsheets, making it difficult to analyze large amounts of data efficiently.

This project helps by:

Centralizing student information.
Automating attendance and marks analysis.
Generating key academic performance indicators.
Providing visual dashboards for easy understanding.
Supporting academic planning and student monitoring.

System Workflow
Student data is collected in Excel format.
Data is imported using Pandas.
Records are stored in an SQLite database.
Data cleaning and preprocessing are performed.
Key Performance Indicators (KPIs) are calculated.
Analytical reports and visualizations are generated.
Results are displayed through an interactive Streamlit dashboard.
